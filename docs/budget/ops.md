# 隐私预算运维手册

## 1. 概述

预算模块为 DP 操作提供 ε/δ 管控。支持内存（默认）与 SQLite 两种后端。

## 2. 配置

| 环境变量 | 默认值 | 说明 |
|---|---|---|
| `PRIVACY_BUDGET_DB` | 空（内存） | SQLite 文件路径 |
| `PRIVACY_BUDGET_WINDOW_SECONDS` | 0 | 自动重置窗口 |

## 3. 预算规划

| 场景 | 建议 ε 总量 | 单次 ε | 可查询次数 |
|---|---|---|---|
| 高隐私（医疗） | 1.0 | 0.1 | 10 |
| 中隐私（金融） | 5.0 | 0.5 | 10 |
| 低隐私（内部） | 10.0 | 1.0 | 10 |

## 4. 故障排查

| 问题 | 原因 | 解决方案 |
|---|---|---|
| BudgetExhaustedException | 预算耗尽 | reset 或增大预算 |
| 预算消耗过快 | 单次 ε 过大 | 降低单次 ε |

## 5. 最佳实践

1. 生产环境使用 SQLite 模式
2. 按业务隔离 namespace
3. 监控预算水位
4. 避免过大 ε（> 5 隐私保护弱）
